# Doações

> Tradução da [versão em inglês](../en/donations.md). O inglês é a
> fonte de verdade; este arquivo o segue.

O Doctrina é gratuito, licenciado MIT e construído com zero telemetria —
ele nunca vai monetizar seus dados. Se ele economiza seu tempo, aqui está
como retribuir.

## Formas de apoiar

| Canal | Onde |
|-------|------|
| **GitHub Sponsors** | [github.com/sponsors/GCarin1](https://github.com/sponsors/GCarin1) |
| **Estrela no repo** | Uma estrela no [GitHub](https://github.com/GCarin1/Doctrina) ajuda genuinamente na descoberta — é a doação de custo zero. |
| **Espalhe a palavra** | Escreva sobre sua experiência de adoção; relatos do mundo real são o que o [protocolo de validação](validation.md) mais precisa. |

> Nota do mantenedor: canais adicionais (Pix, Buy Me a Coffee, Open
> Collective) são configurados em
> [`.github/FUNDING.yml`](https://github.com/GCarin1/Doctrina/blob/main/.github/FUNDING.yml)
> e aparecem no botão "Sponsor" do GitHub quando habilitados.

## Para onde vai o dinheiro

O Doctrina não tem conta de infraestrutura por design — sem servidores,
sem bancos, sem APIs. Doações financiam tempo de manutenção: triagem de
issues, revisão de contribuições de adapters, paridade das docs EN/PT e
o trabalho de validação empírica que mantém o framework honesto.

## Uma doação que não custa nada

Rode o protocolo. Adote o Doctrina num projeto real, rode
`doctrina metrics --save` antes e mensalmente depois, e compartilhe os
números numa issue — positivos ou negativos. Dados honestos são a
contribuição mais valiosa que este projeto pode receber.
